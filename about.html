About Umer Gadgets

Built by a learner, inspired by professional gaming brands.

Our Story

Umer Gadgets is a fictional gaming accessories store created as a frontend web development project. The goal is to learn how modern e-commerce websites are designed using HTML, CSS, JavaScript, Git, and GitHub Pages.

This project focuses on clean design, responsive layouts, and reusable components while following professional development practices.

🎯 Mission

Create fast, modern, and visually appealing websites while continuously improving web development skills.

🚀 Vision

Build professional portfolio projects that demonstrate practical frontend development skills.
